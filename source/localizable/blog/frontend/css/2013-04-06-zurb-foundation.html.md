## Installation

[Ruby Gem](http://rubygems.org/gems/zurb-foundation)

[Gem Instructions](http://foundation.zurb.com/docs/sass.html)

Install

    $ sudo gem install zurb-foundation

Update

    $ sudo gem update zurb-foundation
